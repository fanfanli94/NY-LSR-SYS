DESCRIPTION
This pakage contains all the neccessary files to run our website.
The source code is in the CODE folder and the main files are:
*.js/*.css/*.html - the main UI component of the website
*.csv - the processed data for the website
./assets - icons, images for the website
./markercluster - contains files to render leaflet map markers
./DataProcess_Algorithm - comtains code to generate the processed data.

INSTALLATION
The required software are any software which can launch a local server.
We recommend python, or Live Server VS Code extension.
Visit https://www.python.org/ to download python
Visit extension marketplace in VS Code to install live server extension for VS Code

EXECUTION
Start a server using which ever method you prefer in the CODE folder.

For python:
python -m http.server 8000
Then go localhost:8000 in your browser to view the website.

For Live Server:
Open CODE folder in VS Code. Right click index.html and click "Open with Live Server"
This will open the webpage in your defualt browser.

Feel free to play with the website.
